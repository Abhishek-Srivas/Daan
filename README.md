# Installation
1. Clone repository:
```bash
$ git clone 
```

2. Install dependencies:

```bash
$ npm i
```
3. Open `.env` and edit config, if it's necessary.(Put your atlas uri in backend)

4. Run example:
```bash
npm start
```
### **Note**: cd ./FrontEnd and cd ./Backend, then proceed to step 2.
